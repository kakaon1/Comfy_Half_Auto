# Triton이 설치가 자동으로 안될때.

github에 추가적으로 올린 파일을 받아서 파일이 있는 경로로 이동 후.

"python -s -m pip install triton-3.2.0-cp312-cp312-win\_amd64.whl" 명령어를 사용하여 해결.
